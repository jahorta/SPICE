Purpose
  - This document is a Ghidra annotation guide for the field X-menu opened by pressing X while moving around in field gameplay.
  - The confirmed runtime file is /field/HrsBin_Status.mll, not loose /field/HRSBin.bin.
  - Suggested names are intentionally descriptive. Treat the high-confidence controller, loader, and global names as good candidates for Ghidra labels.
  - Record-level names are tracked in Docs\BinFileRecords.md. This guide focuses on controller functions, status-bank globals, and object/control fields that make those record names understandable in Ghidra.

4. Root selector and stat carousel

Root stat carousel controller
  - Address: 80183d90
  - Current label: FUN_80183d90
  - Suggested label: FieldStatusMenu_UpdateRootStatCarousel_80183d90
  - Role:
    - Handles left/right cycling for the five stat windows in the initial/root party selector.
    - Controller code 7 increments the current slot field and wraps 5 -> 0.
    - Controller code 6 decrements the current slot field and wraps -1 -> 4.
  - Object fields:
    - +0x44: current stat window slot, values 0..4.
    - +0x48: previous stat window slot.
    - +0x14: bottom next-page/back toggle field.
  - Five window roles:
    - 0: Attack, Defense, Will, Magic Defense.
    - 1: Hit %, Dodge %, Quick.
    - 2: weapon and armor equipment.
    - 3: accessory equipment.
    - 4: EXP and next EXP.
  - Evidence:
